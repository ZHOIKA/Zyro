# Sobre o Zyro

O Zyro é uma plataforma moderna para criação e gerenciamento de Rich Presence no Discord.

Com ele, usuários podem personalizar completamente sua presença, adicionando imagens, textos, botões, informações de atividade e muito mais através de uma interface simples e intuitiva.


# Requisitos do Sistema

- Android 8.1 ou superior
- Recomendado: 3GB de RAM ou mais

*(O desempenho pode variar dependendo do dispositivo.)*

# Recursos

- [x] Rich Presence personalizado
- [x] Botões interativos
- [x] Detecção de aplicativos em execução
- [x] Detecção de mídia reproduzida
- [x] Timestamps personalizados
- [x] Status personalizado
- [x] Sistema de presets
- [x] Configurações personalizadas
- [x] Suporte para imagens próprias
- [x] Suporte para GIFs
- [x] Imagens externas por URL
- [x] Visualização do Rich Presence dentro do aplicativo
- [x] Execução em segundo plano
- [x] Interface moderna baseada em Material Design


# Configuração

O Zyro possui uma configuração simples e rápida.

Após instalar, basta conectar sua conta Discord e personalizar sua presença conforme suas preferências.


# Tecnologia

O Zyro utiliza tecnologias modernas para oferecer uma experiência rápida, estável e personalizada.

- Kotlin
- Jetpack Compose
- Material Design
- Android SDK
